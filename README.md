# Web Scraping con Python

Este proyecto consiste en la extracción de datos desde una página web utilizando Python y la librería BeautifulSoup.

# Tecnologías utilizadas

* Python
* BeautifulSoup
* urllib

# Objetivo

Extraer enlaces (`<a href>`) desde una página web.

# Funcionalidad

* Conexión a una URL
* Lectura del contenido HTML
* Parseo del HTML con BeautifulSoup
* Extracción de enlaces
* Limpieza de datos

# Web Scraping vs Web Crawling

* **Web Scraping**: extracción de datos específicos desde una página web
* **Web Crawling**: navegación automática entre enlaces

# Ejecución

```bash
pip install beautifulsoup4
python web_scraping_srt.py
```
